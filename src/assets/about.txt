# Sobre

Jogo de xadrez criado para o trabalho de introdução a programação.

## Descrição
Para este trabalho, desenvolvemos um jogo de xadrez com interface de linha de comando.
A implementação foi feita usando a linguagem C, usando o padrão de linguagem estruturada e bibliotecas nativas da linguagem C.
